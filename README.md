# reverse-polish-notation
example expandable solution for RPN with two operands ("+", "-")

used tools:

- jdk: 1.8.0_161
- Maven 3.3.9
- (for tests) junit 4.12 

## usage: 
1. clone repository
2. mvn compile exec:java -Dexec.args='"12 3 +"' <- REALLY IMPORTANT (expression should be in double quotes)
